# linux Shell 脚本攻略
- 用来记录linux shell 脚本学习的示例与笔记

- 操作系统47~18.04.1-Ubuntu

# 学习之前的配置
- 下面是一些简单的配置

## vim
```vim
#.vimrc
set nu
sy on
set ruler
set smartindent shiftwidth=4
set tabstop=4
set expandtab

set listchars=tab:>-,trail:~
set list
colorscheme desert
```
## shell
```

#.bashrc
set -o vi
alias l='ls-lh'
alias vi=vim

```
